OBJECTIVE

Build a full-stack AI-powered LIMS document processing system that:

CORE AI FUNCTIONAL REQUIREMENTS
========================================================

Document Parsing:
- Extract structured and unstructured content
- Extract tables accurately
- Perform OCR for scanned documents

Intelligent Mapping:
- Determine correct sheet and column placement
- Understand LIMS relationships
- Maintain parent-child dependencies (Example: Components must link to Analysis)

Data Normalization:
- Standardize units
- Normalize numeric formats
- Remove duplicates
- Clean text

Validation:
- Validate required fields
- Validate cross-sheet references
- Validate numeric ranges

Confidence Scoring:
- Assign confidence score to extracted values
- Flag uncertain mappings

Error Handling:
- Continue processing partial documents
- Produce validation report

Logging:
- Maintain detailed audit log
- Record mapping decisions
